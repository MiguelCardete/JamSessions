# Progresiones comunes

## Blues
El blues consta de 12 compases con una estructura determinada.
### Blues simple
En su forma más simple, solo se necesitan 3 acordes <b>dominantes</b> (I III VIIb).


|   |   |  |  |
|:---:|:---:|:--:|:--:|
|  I 7 |  I 7 | I 7 | I 7 |
| IV 7 | IV 7 | I 7 | I 7 |
|  V 7 | IV 7 | I 7 | I 7 |

<b>Ejemplo:</b> Blues en C (Do)

|   |   |  |  |
|:---:|:---:|:--:|:--:|
|  C7 |  C7 | C7 | C7 |
| F7 | F7 | C7 | C7 |
|  G7 | F7 | C7 | C7 |
